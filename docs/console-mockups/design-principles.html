Design Principles & UX

Semantic coloring / Visual hierarchy / Spacing / Iconography / User psychology
1. Semantic Color System
Color = Meaning
Never decorative
Every color communicates something specific. Emerald means passive. Blue means chat. Violet means agent. Amber means warning. Red means failure. No color exists without semantic purpose.
Passive emerald -- listening, storing, human-operated. Mode badges, pipeline, feed dots, heartbeat bars, unread KPI.
Chat cyan-blue -- conversational AI, API-driven, RAG-augmented. Mode badges, queue depth, response latency, feed dots, focus rings.
Agent violet -- autonomous, SDK-driven, multi-turn, tool-using. Mode badges, session state, turn counts, tool calls, feed dots.
Warning amber -- degraded, approaching limits, needs attention soon. Degraded dots, budget warnings, high unread counts, degraded heartbeats.
Critical coral -- disconnected, failed, expired, requires action. Error dots, error rows, auth-expired alerts, failed heartbeats.
Neutral gray -- informational, no urgency. Timestamps, secondary text, neutral KPIs, pipeline neutral nodes.
Three Opacity Tiers
Per mode color
Full (100%) -- text, dots, active indicators. Draws the eye.
Soft (30%) -- badge backgrounds, glow shadows. Suggests association.
Wash (6-12%) -- row highlights, card backgrounds, hover states. Whispers context.
2. Visual Hierarchy
5-Second Rule
Scan pattern
The Soup Kitchen answers "is everything OK?" in under 5 seconds:

1st: KPI strip -- large numbers, color-coded. Red = problem.
2nd: Alert banner -- only visible when something needs you.
3rd: Status dot column -- vertical scan of green/amber/red.
4th: Pipeline column -- mode-specific runtime state.
5th: Activity feed -- peripheral, ambient awareness.

5. User Psychology
Calm by Default
Alarm by exception
The console feels quiet when everything is healthy. Emerald dots, gray timestamps, muted text. The absence of color IS the signal that things are fine. Red and amber only appear when something needs attention.
Progressive Disclosure
Depth on demand
Soup Kitchen: Fleet-wide health at a glance. No detail.
Line row click: One line's full context. 7 tabs of depth.
Tab navigation: Each tab reveals a single concern.
Inline expand: Agent sessions, access entries expand in-place.
Mode = Mental Model
Cognitive wayfinding
When viewing a passive line, the UI shifts to emerald tones. Chat = blue. Agent = violet. Cognitive wayfinding. Switching modes feels like entering a different room.
Data Density
Information not noise
Monospace for data, sans for chrome. The eye scans mono for facts, skips sans for structure.
Align data in columns. The table is the primary pattern.
Ghost text for structure. Headers and labels at t5 orient without competing.
Motion with Purpose
Not decoration
Status dot breathing: Connected lines pulse gently. Confirms liveness.
Staggered page load: Components fade-up with 50ms delays. Systems coming online.
Hover lift on KPI cards: 1px translateY + border highlight. Confirms interactivity.
Feed prepend slide: New events enter with a subtle animation.
No animation on: mode switching, tab transitions, table sorting. Instant and mechanical.
